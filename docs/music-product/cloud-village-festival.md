## 二、会议内容摘录

### Topic 1 - 年度报告

- 共情优先（靠走心吸引用户是网易的天赋），可行性是底线
- 广告体验：新鲜度、分用户行为差异化激励
- 特殊日子纪念弹窗 + 云小编贡献维护相关内容
- 已故歌手纪念账号：君生我未生，我生君已老。已故歌手信息完善及准确 → 歌手人生阶段里程碑 → 粉丝留言

### Topic 2 - 功能拓展

- **音乐探探**：左滑喜欢右滑无感（漫游已实现，但引导可以优化）
- **人声伴奏分轨**：只听伴奏或只听人声
- **歌曲推荐**：降低感受成本（展示歌手、关键词、评论），直接进高潮——视听多感官的延伸
- **八竿子打不着推荐**：多样性，压箱底歌曲翻起来，给音乐人更多灵感，平台资产最大化
- **APP 兼容**：一边听歌一边刷小红书/微博/导航
- **歌曲 MV**：做好 MV 分发

### 其他讨论

- 互联网各平台在用户在线流量不变的情况下，可由竞争转为合作关系，合作兼容完全有可能
- 音乐空窗期体验优化

## 三、一些思考

### 总结 Highlight

> *短期增量视角（场景是否清晰准确）、长期价值视角、行业对比视角都很重要。比如短视频软件都类似，但是音乐软件 UI 都不一样。*

> *接触不一样的人，了解陌生领域的知识，主动提问、思考，寻找能汲取到的共同性。*

> *共创环节多关注过程，大胆地去散发思维、发表观点，既然能认知到本身已经不足了，那就去试着争取一下吧，拥有的总会比原先的自己更多。*
