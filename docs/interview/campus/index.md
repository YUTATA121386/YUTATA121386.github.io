# 校招面试 · 通用高频问题

其实内容都差不多：

- 实习准备这个是第一次实习，做得最全的，记录地很详细，初出茅庐但也算专业

- 曲库面试话术笔记是想投网易的校招，但没用上，初筛就挂了，但基本内容造福了后面几家的面试

- 荔枝这个是加入了对音频这个赛道的一些研究，虽然面试官很和蔼，但方向是活动营收这一块，和我做的不太相关，最后也是一面挂

- 橙果视界和优趣汇就是纯纯垃圾岗，橙果视界的业务面直接让HR来面我，当场就发录用，没啥含金量，实习转正也像画大饼，入职邮件还说我目前还很入门，算不上达标，直接拒了；优趣汇虽然叫品牌运营，但其实就是电商运营，业务面试官直接跟我说和销售没啥区别，发展方向不一样，最后不了了之。

- 昆仑天工就很抓马了，其实算是社招也算是校招，工作内容和我在杭州网易的Leader差不多，要管30人体量的外包＋实习生，一面通过后HR都开始跟我介绍股票激励了，本来以为二面也能过的，结果产品总监面给我上压力，虽然也不算稀烂，但最后HR告知优先让实习一年的人转正了，玛德。

- 东西世界是个体量小生意大的公司，其实可能是岗位大部分偏执行的缘故，所以一开始和COO电话面了17分钟，业务问题是关于简历里的三次审核流程迭代的，其他就是兴趣爱好、稳定性和生活方面的问题，约了视频面试是线上，面试阵容很顶配，CEO＋COO＋Leader，主要还是CEO在问，但业务问题也不多，像是行业沙龙，聊聊天，当天就发了Offer
