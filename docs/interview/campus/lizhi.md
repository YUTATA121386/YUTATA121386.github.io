## 小陪伴产品认知

主打的营收模式是"情感 + 音乐"的声音社交平台。

| 付费点 | 形式 |
|--------|------|
| 礼物打赏 | 用户打赏可选主播爆音、自我介绍等 |
| 会员增值 | 订阅、专属装扮、进场特效，满足身份社交需求 |
| 专属付费场景 | 游戏陪玩、付费点唱、付费门槛房间 |
| 站内赛事 | 顶流歌王等 IP 赛事，通过艺人流量扶持达成商业合作 |

**会员/权益核心逻辑**：用"专属特权"提高用户粘性，打造付费分层。

**竞品**：TT 语音、PP 语音、小西米语音、带带陪玩，以及 Soul、网易 LOOK 直播等音频陪伴赛道产品。
