# 关于

::: tip 关于这本手帖

把所见、所闻、所思，写进一本手帖里。

:::

---

**YUTATA 知识档案** 是一个私人知识花园。这里没有观众，只有记录。

### 五个栏目

| 栏目 | 定位 | 状态 |
|------|------|------|
| 上升清单 | 成长轨迹、自我剖析、关键决策 | 🚧 |
| 通识积累 | 影视评级、音乐鉴赏、烹饪技法、行业知识 | ✅ |
| 圣地巡礼 | 旅行记录、取景地打卡、文化见闻 | 🚧 |
| 云音乐产品学习 | 产品拆解、行业分析、功能思考 | ✅ |
| 校招面试话术 | 面试记录、话术复盘、经验沉淀 | ✅ |

### 写作哲学

- **费曼原则** —— 如果写不清楚，就是没想清楚
- **长期主义** —— 积累五年、十年，不做速成品
- **真诚记录** —— 不为表演，只为自己
