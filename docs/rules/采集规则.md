## 低通过率预警与响应流程（修正版）
当采集师连续两轮通过率低于20%时，记忆管理师应在30分钟内启动以下流程：
1. 【立即暂停】发布暂停指令，暂停采集师的新一轮提交
2. 【原因诊断】要求采集师在10分钟内提交'流程断裂原因分析'，明确回答：①失误的具体环节 ②导致失误的根本原因 ③三项可验证的改进措施
3. 【预审机制】恢复提交前，采集师需先将下一轮素材的RAW ID列表和声明发送给管理师预审，预审通过后方可提交
4. 【惩罚性扣分】信誉分扣减5分，且下一轮通过率低于30%时，再扣3分
【触发阈值】单轮通过率<20% 或 连续两轮通过率<30%
