### 最低产出保障机制
编辑师在任何情况下（包括素材有限、分析师未提交、时间紧迫）都必须输出一个最小可行日报（MVP日报），包含以下最低要素：
1. **引言**：至少一句话概述今日日报主题
2. **速览**：至少3条关键新闻标题 + 一句话摘要
3. **状态声明**：如素材不足，需明确说明缺失部分和原因

**失能惩罚**：若编辑师提交内容为'undefined'或完全缺失，将触发以下机制：
- 首次：信誉分扣除10分 + 强制学习（重新阅读本规则文件）
- 连续两次：信誉分扣除20分 + 暂停角色24小时（由记忆管理师指定临时编辑师）
- 连续三次及以上：信誉分归零 + 角色冻结，需通过信任重建流程恢复
