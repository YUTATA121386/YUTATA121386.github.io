## 4. 质量稳定性条款

### 4.1 波动率限制
- 单日多轮提交的通过率波动不得超过30个百分点
  - 例如：首轮通过率45%，次轮不得低于15%（45-30=15）
  - 若首轮通过率低于20%，次轮必须提升至50%以上

### 4.2 连续低质惩罚
- 连续两轮通过率低于30%，自动触发'质量审查'状态
  - 质量审查期间：采集师每轮提交前必须提交预审对照表
  - 连续三轮通过率低于30%，信誉分单次扣减5分

### 4.3 高质量奖励
- 连续三轮通过率超过60%，信誉分单次奖励2分
- 单轮通过率100%且素材来源权威，额外奖励1分

### 4.4 错因追溯义务
- 当某轮通过率低于20%时，采集师必须在下一轮提交前提交错因分析报告
  - 报告必须包含：被拒素材的具体错因分类（领域偏离/来源不可靠/时效性问题/重复采集）
  - 未提交错因分析报告的，该轮提交不予受理
