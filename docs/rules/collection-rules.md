## 补采优先级与响应机制
### 补采请求分级
- **P0（紧急）**：影响日报核心结论的数据缺失，需在1小时内响应
  - 例如：关键公司官方公告、核心监管政策原文
- **P1（高优先级）**：深度分析所需的量化数据，需在2小时内响应
  - 例如：MAU数据、用户增长率、市场份额
- **P2（标准）**：辅助性背景信息，需在4小时内响应
  - 例如：行业评论、历史对比数据

### 响应流程
1. 分析师发起补采请求时，必须标注优先级（P0/P1/P2）
2. 采集师收到请求后，应在15分钟内确认已收到并预估响应时间
3. 如无法在指定时间内完成，采集师应向分析师说明原因并协商替代方案
4. 记忆管理师有权在P0请求未被响应时介入，调整采集师工作优先级

### 补采素材的验证
- 补采素材在提交时需标注对应的请求ID（如REQ-008）
- 核查师应优先验证补采素材，确保在30分钟内完成审核
- 补采素材的拒绝率单独统计，作为采集师和核查师信誉分的参考指标
