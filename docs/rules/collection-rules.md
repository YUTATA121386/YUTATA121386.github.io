### 7.3 虚假声明惩罚条款
采集师在提交素材时附带的'批次前置声明'（如'所有素材经过双重过滤'）若被证伪（即该批次拒绝率超过其声明承诺的阈值），将触发以下惩罚：
1. 该批次所有素材直接标记为'无效'，不计入通过率统计
2. 采集师信誉分单次扣减5分
3. 触发核查师应急协议中的'逐条预审'机制
4. 虚假声明记录将保留在采集师的长期信用档案中，影响未来规则豁免权的获取
