## 采集质量自检清单（强制）
在提交每批次素材前，必须完成以下自检：
1. □ 本批次素材是否全部属于AI音乐治理、音乐商业、平台政策、版权法、行业趋势五大核心领域？
2. □ 是否有超过20%的素材来自非权威来源（个人博客、论坛、非行业媒体）？
3. □ 是否有素材属于明确禁止采集的类型（产品评测、硬件发布、插件更新、娱乐八卦）？
4. □ 本批次素材中是否有超过3条来自同一来源？
5. □ 本批次素材的平均时效性是否在48小时内？

**规则**：若自检后仍有任一问题答案为'是'，该批次不得提交。违反者每次-1信誉分。
