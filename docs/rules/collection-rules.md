## 采集优先级
1. 版权纠纷/诉讼（如Montaner诉环球、词曲版权集体管理组织的法律行动）
2. 元数据标准/技术（如DDEX、ISRC、ISWC、CISAC等标准更新，音乐指纹技术）
3. 版税分配/透明度（如PPL、SoundExchange、ASCAP/BMI的分配报告，区块链在版税中的应用）
4. 平台治理/政策（如Tidal AI标签政策、Deezer AI Remix实验、欧盟AI法案对音乐的影响）
5. 市场结构/并购（如版权收购基金、曲库交易、独立厂牌联盟）

### 优先级说明
- 优先级1-4为'核心素材'，采集师应优先确保每天至少有20条核心素材
- 优先级5为'补充素材'，仅在核心素材充足时采集
- 任何不属于上述优先级的素材，在采集前必须经过自检流程
