## 核心领域定义（白名单/黑名单操作手册）
采集师必须严格聚焦于'曲库行业'（Music Catalog Industry）。以下为明确的白名单和黑名单，如有疑问，优先参考白名单。

### ✅ 白名单（可采集）
1. **音乐版权交易**：
   - 曲库收购/出售公告（如：Hipgnosis、Round Hill、Reservoir、Primary Wave的交易）
   - 版权预付款/融资（如：Kobalt、Lyric Financial的预付款业务）
   - 公司并购（如：Sony收购AWAL、BMG收购）
2. **版权资产管理**：
   - 版税分配与纠纷（如：Spotify版税改革、Meta与华纳纠纷）
   - 版权侵权诉讼与判赔（如：Triller判赔案、AI版权诉讼）
   - 版权登记与确权（如：美国版权局政策、MLC运营）
3. **技术赋能**：
   - AI音乐生成与版权（如：Suno、Udio的训练数据争议）
   - 音乐技术标准（如：DDEX、ISRC、ISWC相关更新）
   - 区块链/NFT在音乐版权的应用
4. **行业基础设施**：
   - 表演权组织（如：BMI、ASCAP、SESAC的动态）
   - 机械复制权组织（如：MLC、SoundExchange）
   - 集体管理组织改革
5. **政策与法律**：
   - 版权法修订
   - AI监管政策（如：欧盟AI法案对音乐的影响）
   - 反垄断调查

### ❌ 黑名单（禁止采集）
1. **艺人/音乐人个人新闻**：
   - 新专辑发布、巡演信息、获奖感言、个人生活
   - 除非该新闻直接涉及版权交易（如：Taylor Swift重录专辑的版权归属争议）
2. **纯音乐评论/榜单**：
   - Billboard榜单分析、乐评、年度最佳专辑评选
3. **与音乐版权无关的AI通用技术**：
   - 通用大模型发布（如：GPT-5、Claude 4）、AI绘画、AI视频
   - 除非该技术直接应用于音乐版权管理
4. **演出/娱乐产业**：
   - 演唱会票房、音乐节策划、演出场馆动态
5. **硬件/设备**：
   - 乐器发布、音频设备评测、录音棚技术
