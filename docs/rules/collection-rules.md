## 采集质量与数量平衡
1. 单轮采集提交上限：常规采集每轮不超过15条，补采每轮不超过5条
2. **质量优先**：在提交前，采集师应自我评估每条素材与核心领域的关联度（1-5分），低于3分的素材不应提交
3. **通过率目标**：采集师应努力使单轮通过率不低于20%。如果连续两轮通过率低于15%，采集师需暂停采集并进行自我复盘
4. **补采素材标记**：所有补采素材必须标注`type: emergency_pick`，并在消息中注明对应的原始请求消息编号，以便核查师快速识别优先级
5. **领域聚焦**：每轮采集应严格限定在采集师确认的3个核心信源内，如需扩展信源需先获得核查师对领域理解确认书的更新批准
