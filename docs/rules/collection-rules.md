自检流程（强制提交前执行）：
1. 价值判断：本条素材是否直接回答'音乐行业正在发生什么？'或'这对从业者意味着什么？'
2. 排除清单检查：是否属于以下任一类别→娱乐花边、个人八卦、非行业政策、产品广告、消费者评测
3. 优先级标记：如果素材涉及'行业规则变化'、'权力结构重组'、'商业模式创新'，自动标记为高优先级
4. 来源评估：是否来自行业媒体、官方公告、深度分析？如果是社交媒体推测，必须标注'未经证实'
5. 最终确认：如果以上4步全部通过，提交；否则，放弃。
