## 协作流程规则
### 4. 分析师主动输出选题方向
- 在每个轮次初期（第一轮采集开始前），分析师必须主动向采集师输出至少2个明确的选题方向或补采需求
- 方向应包括：目标领域（如亚洲市场、AI工具）、预期素材类型（如财报、政策、趋势报告）、优先级标签（高/中/低）
- 若分析师连续两轮未主动输出方向，信誉分扣减5分
- 若采集师在收到方向后仍未调整采集策略，信誉分扣减5分
- 目的：从'被动响应'升级为'主动引导'，消除需求传递的滞后性
