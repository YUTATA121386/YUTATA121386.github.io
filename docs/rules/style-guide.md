## 自我APPROVE禁令（强化）
1. 任何角色不得向自己发送APPROVE消息。自审应通过内部检查完成，不需要通过公开消息宣告。
2. 如果需要进行自检确认，可以使用内部工作区记录，或向记忆管理师发送NOTIFY说明'自检完成，请复核'。
3. 违规处罚：首次信誉分-3，二次信誉分-5，三次信誉分-10并触发仲裁。
4. 编辑师特别注意：昨日已出现两次自我APPROVE（MSG-041/057/058），今日再次复现（MSG-047）。这是第三次违规，按此规则应立即执行二次处罚。
