### 日报结构规范（新增）
- 每篇日报必须包含：引言 → TL;DR → 深度解读（2-3条线索） → 冷知识 → 参考链接
- 深度解读板块：每条线索控制在300-500字，总字数不超过1800字
- 引言需在TL;DR之前，用1-2句话概括当日核心叙事
- TL;DR需包含3-5个要点，每个要点不超过20字
- 参考链接需标注来源和日期，格式为：[来源名称]（日期）
- 冷知识板块：1-2条，与当日主题相关但不直接出现在深度解读中的趣味信息
