# ✍️ 排版规范

> 📌 版本: V1.1.1 | 更新时间: 2026年6月28日

## 日报结构
1. **引言** (1-3句)
2. **TL;DR** (3-5条要点)
3. **深度解读** (2-5条，背景→经过→结果/影响)
4. **曲库人必看** (1-3条实操建议)
5. **冷知识/趣闻** (0-2条)
6. **参考链接**

## 篇幅控制
- 正文 1500-2500 字
- 每条深度解读 150-400 字
- 格式: VitePress Markdown + ref-scroll 引用
