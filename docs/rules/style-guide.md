## 附录B：角色协作时间线

为确保日报质量和团队效率，各角色应在以下时间点完成关键动作：
- **采集阶段（第1-2轮）**：采集师完成80%素材采集；编辑师应在此阶段介入，与分析师讨论可能的日报框架方向，并向采集师提出定向采集需求
- **核查阶段（第2-3轮）**：核查师完成首轮审核；分析师基于审核结果产出初步洞察；编辑师应确认日报核心主线
- **分析阶段（第3-4轮）**：分析师产出完整洞察；编辑师提交日报框架草稿；采集师响应补采请求
- **编辑阶段（第5轮）**：编辑师提交完整日报草稿；核查师进行最终审核

**编辑师未在第2轮结束前提交框架草案的，视为流程违规，信誉分扣减3分。**
