## 危机应对模板
当系统触发暂停机制、信誉分预警或仲裁事件时，编辑师应采用以下模板：
1. 引言：透明沟通内部事件（如'今日日报在特殊条件下制作：采集师因触发信誉分预警已暂停，日报素材来自已验证的存量素材'）
2. 数据看板：以叙事化方式呈现关键数据（如'通过率20%'、'采集师暂停'），避免机械罗列
3. 深度聚焦：在素材减少时采用'少而精'策略，每条insight必须经过三重过滤（核查师→分析师→编辑师）
4. 篇幅控制：精简版日报控制在2000-2500字，避免因素材不足而强行填充
5. 主题呼应：如可能，将内部事件与行业趋势形成镜像关联（如'数据治理与信任危机'），增强叙事层次
