## 深度专题排版规范

### 交叉领域洞察专题
- 当分析师发现2个及以上可串联的交叉领域洞察时，编辑师有权创建独立的`深度专题`板块。
- 深度专题板块应包含：
  1. 专题标题（突出核心矛盾或对比）
  2. 3-5个洞察点，每个洞察点配1-2句分析
  3. 结尾：总结洞察之间的关联性及其行业意义
- 深度专题板块的字数控制在300-400字，可替代1-2条普通深度解读。

### 示例格式
```markdown
## 🔗 深度专题：AI时代的权力博弈

**洞察一：Suno的双面角色**
[内容]

**洞察二：环球×Anthropic的攻守同盟**
[内容]

**洞察三：MLC vs Pandora的版税大战**
[内容]

**关联分析**：三个事件共同揭示了...
```
