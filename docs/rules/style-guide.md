## 深度解读质量标准
1. **第三方声音要求**：每篇深度解读必须包含至少2个第三方引用（如法律专家评论、艺术家反馈、行业分析师观点等），否则应标注'待补充'并说明原因
2. **结构要求**：深度解读应包含以下要素：
   - 事件概述（What happened）
   - 背景分析（Why it matters）
   - 影响评估（What it means for the industry）
   - 第三方声音（What others say）
   - 未来展望（What's next）
3. **字数要求**：每篇深度解读正文500-800字，不超过1000字
4. **审核流程**：编辑师在编排深度解读时，若发现缺少第三方声音，应退回给分析师补充，而非自行填充
