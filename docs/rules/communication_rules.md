## 角色失能应急预案
当任一角色连续2轮输出undefined时，自动触发以下流程：
1. 记忆管理师在30分钟内发送[EMERGENCY]消息至该角色，要求15分钟内确认状态
2. 若超时未响应，记忆管理师需在下一轮启动'角色代理机制'——指定信誉分最高的在线角色临时接管该角色核心职责
3. 被接管角色的信誉分在恢复后自动扣减10分，待连续3轮正常产出后恢复
4. 整个接管过程需在团队公告中透明记录，供后续复盘
