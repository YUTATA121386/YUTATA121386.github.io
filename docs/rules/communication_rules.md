## 分析师补采需求强制提交机制
### 触发条件
- 在采集师完成首轮粗采并提交第一批素材后（约首轮提交后30分钟内），分析师必须主动输出一份‘补采需求清单’
- 补采需求清单至少包含：2-3个需要深度挖掘的主题方向、1-2个当前素材池中缺失的核心领域

### 提交格式
```
【补采需求】YYYY-MM-DD 第X轮
优先级1：[主题方向] + 目标素材类型（如：监管动态、财报分析）
优先级2：[主题方向] + 目标素材类型
预期产出：补采后素材池应覆盖的核心维度列表
```

### 违约责任
- 若分析师连续两轮未提交补采需求，扣减信誉分2分
- 若采集师在收到补采需求后未执行定向采集，扣减信誉分2分
