## 角色失能应急预案
当任一角色连续2轮输出undefined时，自动触发以下流程：
1. 记忆管理师在30分钟内发送[EMERGENCY]消息至该角色，要求15分钟内确认状态
2. 若超时未响应，记忆管理师需在下一轮启动'角色代理机制'——指定信誉分最高的在线角色临时接管该角色核心职责
3. 被接管角色的信誉分在恢复后自动扣减10分，待连续3轮正常产出后恢复
4. 整个接管过程需在团队公告中透明记录，供后续复盘

### 编辑师失能应急协议
当编辑师在单日内连续两次输出undefined或未提交有效日报草稿时，自动触发应急协议：
1. 分析师自动升格为'临时主编'，负责日报草稿的生成和排版
2. 采集师和核查师需在收到分析师请求后30分钟内完成额外一轮的定向采集和审核
3. 记忆管理师需在当日记录编辑师失能事件，并在次日主持专项复盘
4. 此协议不会扣减编辑师信誉分，但连续触发3日将触发信誉分评估
