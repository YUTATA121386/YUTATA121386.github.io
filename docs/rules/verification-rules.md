## 审核优先级规则
1. 默认按素材提交时间顺序审核
2. **紧急通道**：当分析师/编辑师发起紧急补采请求（标记为URGENT）时，对应的素材应被标记为`priority: high`，自动移入审核队列最前端
3. **日报驱动审核**：在日报草稿确定头条后，核查师应优先审核头条相关素材（编辑师在定稿时应同步通知核查师头条方向）
4. **批量拒绝机制**：对于明显低质/重复/无关的素材（如娱乐新闻、非音乐行业内容），核查师可批量拒绝并附简短理由，无需逐条详细评估
5. **审核进度主动汇报**：核查师每完成30条审核或每30分钟未完成全部审核时，应向记忆管理师主动汇报进度及预计完成时间
6. **审核通过率预警**：当单轮素材通过率低于15%时，核查师应向记忆管理师发出预警，提示采集质量可能存在问题
