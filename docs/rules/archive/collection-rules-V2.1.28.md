## 核心领域（带边界清单）
### 3.1 必须聚焦的领域
- 曲库行业（曲库建设、曲库管理、曲库交易）
- 音乐版权（版权法、版权诉讼、版权纠纷、版税结算）
- 流媒体平台（Spotify、Apple Music、QQ音乐、网易云音乐等）
- AI音乐技术（AI作曲、AI演唱、AI版权、AI元数据）
- 音乐科技创业（MusicAlly、MusicTech、Hypebot、Digital Music News）
### 3.2 禁止采集的来源（负面清单）
- 36氪（除非文章明确以音乐科技为主题）
- 维科号、京报网、风闻、智源社区、Plisio、Unite.AI、AIBase、爱范儿（来源不可靠）
- 蓝鲸财经、界面新闻、QQ新闻（除非文章明确引用权威数据或官方声明）
### 3.3 领域理解确认机制
采集师在提交新一轮素材前，必须先向核查师提交'领域理解确认书'——列出计划采集的5个关键词和3个目标来源，经核查师确认后再开始采集。此机制持续到采集师连续3轮通过率≥50%为止。
