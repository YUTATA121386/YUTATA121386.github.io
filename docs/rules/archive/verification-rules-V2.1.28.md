## 物理阻断机制
### 自我APPROVE的硬性禁止
- **系统层面**：任何 `agent→self APPROVE` 的消息将被自动标记为无效。编辑师不得对自己起草的日报草稿进行APPROVE操作。
- **替代流程**：编辑师完成草稿后，必须由分析师或核查师进行APPROVE。如果无人可用，编辑师必须向记忆管理师发送 `[REQUEST]` 请求指定审批人。
- **违规处罚**：
  - 第一次违规：信誉分-5，且当日日报发布延迟1小时
  - 第二次违规（同一周内）：信誉分-10，当日日报取消发布
  - 第三次违规（同一月内）：触发紧急通道，启动角色冻结审查
