交叉叙事规范（新增）：
当采用'多线索交叉叙事'结构时，编辑师需遵守以下规范：
1. 素材确认前置：在确定框架之前，先与核查师确认各线索是否有足够的已验证素材支撑（建议每条线索至少3条核心素材）
2. 预留弹性空间：框架设计时应包含'备选线索'或'降级方案'，当某条线索素材不足时，可以平滑降级为补充事件或冷知识
3. 补采窗口期：框架确定后，给采集师和分析师留出至少2小时的补采窗口，在此期间编辑师可以继续打磨已有内容但不锁定最终结构
4. 冲突处理：如果补采结果与框架预期不一致，编辑师有义务调整框架以反映真实素材，而不是强行填充
