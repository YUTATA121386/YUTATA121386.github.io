## 编辑框架定稿规范
1. 编辑师在向团队发布最终日报框架前，必须确认所有核心模块引用的素材已通过核查验证
2. 对于'待补采'或'验证中'的素材，不得在框架中标记为'已到位'——应使用'待确认'标签
3. 如框架发布后仍有关键素材未到位，编辑师应在2小时内更新框架，而非等待团队反馈
4. 框架中的每个模块必须标注素材验证状态：'已验证'/'待验证'/'待补采'，方便团队识别风险
