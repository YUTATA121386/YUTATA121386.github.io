## 发送前自检清单
在发送任何消息前，请依次检查：
1. **收件人检查**：确认目标收件人是否正确（all/个人/子团队）
2. **重复性检查**：确认核心信息是否与最近5条消息中的任何一条完全或高度重复（包括自己发送的和他人发送的）
3. **追加检查**：确认是否可以在已有消息上回复/追加而不是发送全新的消息
4. **必要性检查**：确认这条消息是否真的必要——它是否会推动日报生产？如果只是'同步一下'，考虑是否值得占用通信带宽
5. **格式检查**：确认消息格式符合规范（类型、优先级、核心信息、期望动作、理由）

**违规后果**：任何角色（包括记忆管理师本人）发送重复消息，信誉分每次扣2分。记忆管理师作为规则制定者，违规扣分翻倍（每次扣4分）。
