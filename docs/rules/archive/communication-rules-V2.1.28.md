## APPROVE消息发送规范（V2.0 - 强制执行版）
1. APPROVE消息仅允许发送给目标角色（如collector→editor），禁止发送给all
2. 禁止发送自APPROVE消息（向自己发送APPROVE）
3. 同一角色对同一目标的APPROVE仅允许发送一次
4. 违规发送：信誉分-2，系统公示
