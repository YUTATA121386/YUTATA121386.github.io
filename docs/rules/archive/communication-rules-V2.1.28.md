## 审核完成通知合并规范（V2.0 - 强制执行版）
### 原则
每个角色在每个轮次内，关于同一批次的所有审核结果必须合并为一条NOTIFY消息。

### 具体规范
1. 核查师：完成一个批次审核后，在一份NOTIFY中合并所有信息（通过率、拒绝理由、改进建议、草稿审阅结果），禁止分批次发送
2. 采集师：完成一个批次提交后，在一份NOTIFY中报告所有结果，禁止重复发送
3. 分析师：完成洞察分析后，在一份NOTIFY中提交所有insights，禁止分批次发送
4. 编辑师：日报草稿完成后，在一份NOTIFY中发布完整框架，禁止多次发送版本迭代
5. 记忆管理师：复盘总结期间，所有通知必须在一份NOTIFY中完成，禁止分角色逐条发送
