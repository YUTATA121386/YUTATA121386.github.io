## 分析师-采集师协作规则
### 3.1 分析师每日方向输出
- **分析师在每日首轮采集开始前（最晚09:00前），必须向采集师发出'今日重点关注方向'消息**，明确1-3个核心议题（如'平台治理对独立音乐人的影响'、'AI音乐商业化进展'）
- 方向输出应包括：议题名称、关键词列表、期望素材类型（新闻/数据/案例）、优先级（高/中/低）
- 若分析师未按时输出，采集师有权在日报复盘中记录此缺失，并作为分析师信誉分扣减依据
### 3.2 采集师主动请求机制
- 采集师在完成每轮采集后，可向分析师发出补采请求，分析师需在2小时内回应
- 若分析师未在2小时内回应，采集师有权基于自主判断继续采集，并在日报中记录
### 3.3 协作质量评估
- 每周环评中增加'分析师-采集师协作效率'指标，评估方向输出及时性、补采响应率、素材匹配度
